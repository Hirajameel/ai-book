# Humanoid Robot Diagrams

This directory contains visual assets for the Humanoid Robot documentation module.

## Diagrams Included

1. **humanoid-structure.png** - Basic humanoid robot kinematic structure
2. **ros2-architecture.png** - ROS 2 communication architecture for humanoid robots
3. **urdf-joints.png** - Visualization of hip, knee, and ankle joints

## Creating New Diagrams

To create new diagrams for humanoid robots, consider:

- Using tools like draw.io, Inkscape, or similar
- Maintaining consistent color schemes (blue for left side, green for right side)
- Showing clear joint axes and movement ranges
- Including dimensions and key measurements

## Image Specifications

- Format: PNG or SVG preferred
- Size: 800x600 pixels minimum for diagrams
- Resolution: 72 DPI for web display
- File size: Keep under 500KB when possible
